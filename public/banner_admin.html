Como funciona

Este painel salva os banners na coleção banners do MongoDB. Cada banner tem um id fixo por posição. No site, basta ler esses docs e renderizar quando active=true.

Coleção

banners

Campos

  • active (true/false)
  • imageUrl (link direto da imagem)
  • linkUrl (para onde clica)
  • alt (texto alternativo)

Dica

Se o banner estiver desativado, o site deve esconder o slot correspondente.
