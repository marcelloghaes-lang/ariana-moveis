Criar Categoria

Se você escolher uma categoria pai, esta vira subcategoria (parentId + level=1).

Estrutura salva (compatível com seu admin atual)
categories/{docId}:
{
  name, description, slug,
  parentId: null | "idDaCategoriaPai",
  level: 0 | 1,
  active: true|false,
  order: number,
  createdAt, updatedAt
}
